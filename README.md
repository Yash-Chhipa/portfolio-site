# ✦ Yash's Portfolio

A creative, editorial-style portfolio website built to showcase my journey as a developer, open-source contributor, and lifelong learner.

Unlike traditional developer portfolios, this project focuses on storytelling, personality, and memorable experiences rather than simply listing technologies and achievements.
